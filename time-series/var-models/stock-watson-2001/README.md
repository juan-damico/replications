# Stock & Watson (2001)

This folder contains an **R-based replication** of a VAR tutorial inspired by Stock & Watson (2001), with a focus on implementation, interpretation, and reproducibility.

## Objective
The goal is to replicate the main workflow of vector autoregressive (VAR) models in R and document each step clearly.

## Contents
- Data preparation
- VAR estimation
- Lag selection
- Impulse-response analysis
- Discussion of results

## Language
This replication is implemented in **R**.
